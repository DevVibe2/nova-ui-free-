## 🚀 How to Use
 
1. Open `index.html` in your browser
2. Click **"Copy Code"** on any component
3. Paste it into your project
4. Customize colors using CSS variables
 
```css
:root {
  --accent: #6c5ce7;   /* Change to your brand color */
  --grad-1: #6c5ce7;   /* Gradient start */
  --grad-3: #fd79a8;   /* Gradient end */
}
